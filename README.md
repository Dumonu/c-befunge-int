# C Befunge Interpreter

## Supported Commands

| Command | Description                               |
| ------- | ----------------------------------------- |
| +       | pop a, b and push a + b                   |
| -       | pop a, b and push b - a                   |
| *       | pop a, b and push a * b                   |
| /       | pop a, b and push b / a                   |
| .       | pop numerical value from stack and print  |
| ,       | pop ascii value from stack and print      |
| @       | halt                                      |
| 0...f   | push numerical value                      |
